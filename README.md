# WirVSCorona

## What it does
Basierend auf einfachen Selbstauskünften wird anhand einer Simulation vorgeführt, wie sich das eigene Verhalten auf die Umwelt auswirkt.

## Setup:
```
npm i
```
```
ng serve
```
